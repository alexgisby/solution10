Welcome to Solution 10 Happy to have you here.

What is Solution 10?

First and foremost, Solution 10 is NOT a framework. It's a collection of framework-agnostic, well tested classes that can be dropped into any project with minimal hassle and just used.

The goals of Solution 10 are threefold:

  1. Only provide genuinely useful classes. Not the same Validation and Request classes you'll find in every framework worth it's salt.
  2. Well documented, well tested. Docblocks, HTML reference and as close to full Unit test coverage as possible.
  3. To be interoperable, and entirely framework agnostic. Symfony, Zend, Kohana, Lithium, Cake, whatever. Any framework can drop in and go.

How can I install it?

Three ways, via git and GitHub, using Composer or by simply downloading from GitHub's downloads page.

The library itself is all PSR-0 compliant, meaning any modern autoloader can handle it automatically.
